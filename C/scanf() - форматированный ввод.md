`sacnf()` читает данные из stdin потока преобразуя их в значения типа данных C в зависимости от модификатора 
```c
char f, s;
int res = scanf("%c %c", &f, &s);  // res содержит количество считанных данных. в данном случае 2.
```
Можно использовать разделитьельные символы
```c
int res = scanf("%c; %c", &f, &s);  // ; и пробел будут разделителями между символами
```
